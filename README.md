# VEGFA viewers

Interactive 3D structure viewers comparing Evo2-7B / Carbon-3B/8B / ProtGPT2 on VEGFA generation. Hosted on GitHub Pages.

- `index.html` — landing
- `upstream-orf.html` — Evo2 upstream ORF experiment
- `cds.html` — Evo2 CDS continuation
- `carbon-protgpt2.html` — Carbon-8B + ProtGPT2 protein experiment

Each viewer is fully self-contained (PDBs embedded inline). 3D rendering via [3Dmol.js](https://3dmol.org/).
